# SQL Analysis Results

## Dataset Summary

- Total Records: 9694
- Database: MySQL
- Dataset: Superstore

## Key Findings

### Sales Analysis

- Technology generated the highest sales.
- Furniture and Office Supplies followed.

### Regional Analysis

- West and East regions generated the highest revenue.
- Central region generated comparatively lower sales.

### Customer Analysis

- Top customers contributed significantly to total revenue.
- Repeat customers were identified using GROUP BY.

### Product Analysis

- Top-selling products were identified.
- Product quantities were aggregated.

### Profit Analysis

- Some products generated negative profit.
- High discounts reduced profitability.

### Data Quality

- Duplicate Order IDs were identified.
- NULL values were checked.
- Row counts were validated.
